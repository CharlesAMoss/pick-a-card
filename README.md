# _Pick a Card_
## _3/20/17_
### by [Charles Moss](https://twitter.com/CharlesMoss)

## Description
_User will be able to pick will be asked to draw a card_

## Setup
Clone Repo, with node and npm installed go to the cloned folder:

```
$ npm install
```

To run:

```
$ node index.js
```
